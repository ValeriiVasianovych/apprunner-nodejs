Node.js Application

This is a simple Node.js application built with Express.js. It demonstrates a clean, minimalist design with a black and white color scheme.

The application features:

  • Express.js server with routing
  • Static file serving
  • MVC architecture with controllers
  • Responsive design
  • Clean and modern UI
